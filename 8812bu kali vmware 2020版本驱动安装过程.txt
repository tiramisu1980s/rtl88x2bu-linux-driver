8812开启监听方法如下：监听模式开启后桌面右上角是搜不到无线的
启动监控模式：

ip link set wlan0 down
iwconfig wlan0 mode monitor
ip link set wlan0 up

启动托管模式：

ip link set wlan0 down
iwconfig wlan0 mode managed
ip link set wlan0 up
